# 算法题：二叉树根节点到叶子结点路径和等于目标值

## 一、题目描述

给定一个二叉树和一个目标和 `targetSum`，找出所有从**根节点到叶子节点**路径和等于 `targetSum` 的路径，返回这些路径。

**叶子节点**：指没有子节点的节点。

### 示例

```
输入：root = [5,4,8,11,null,13,4,7,2,null,null,5,1], targetSum = 22

       5
      / \
     4   8
    /   / \
   11  13  4
  /  \    / \
 7    2  5   1

输出：[[5,4,11,2], [5,8,4,5]]

解释：
路径 5→4→11→2 的和为 22
路径 5→8→4→5 的和为 22
```

---

## 二、解题思路

### 核心思路：前序遍历 + 回溯

**为什么用前序遍历？**
- 前序遍历顺序：根→左→右
- 从根节点出发，沿着每条路径走到叶子节点
- 天然符合"从根到叶子"的路径要求

**为什么要回溯？**
- 一条路径走完后，需要回到父节点，走另一条分支
- 回溯就是"撤销选择"，恢复路径和状态

---

### 算法步骤

```
1. 从根节点开始前序遍历

2. 对于当前节点：
   a. 将当前节点值加入路径
   b. 更新剩余目标和：targetSum -= node.val

3. 判断是否是叶子节点：
   - 如果是叶子节点且剩余和为 0：找到一条有效路径，加入结果集
   - 如果不是叶子节点：继续递归左右子树

4. 回溯：
   - 从路径中移除当前节点
   - 恢复剩余目标和（可选，因为传递的是新值）

5. 返回所有有效路径
```

## 三、代码实现

### Java 实现

```java
/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode() {}
 *     TreeNode(int val) { this.val = val; }
 *     TreeNode(int val, TreeNode left, TreeNode right) {
 *         this.val = val;
 *         this.left = left;
 *         this.right = right;
 *     }
 * }
 */
class Solution {
    // 结果集：存储所有有效路径
    List<List<Integer>> result = new ArrayList<>();

    // 当前路径：存储遍历过程中的路径
    LinkedList<Integer> path = new LinkedList<>();

    public List<List<Integer>> pathSum(TreeNode root, int targetSum) {
        dfs(root, targetSum);
        return result;
    }

    /**
     * 前序遍历 + 回溯
     * @param node 当前节点
     * @param remainSum 剩余目标和（还需要凑多少）
     */
    private void dfs(TreeNode node, int remainSum) {
        // 边界条件：空节点直接返回
        if (node == null) {
            return;
        }

        // 1. 做选择：将当前节点加入路径
        path.add(node.val);
        remainSum -= node.val;  // 更新剩余和

        // 2. 判断是否是叶子节点
        if (node.left == null && node.right == null) {
            // 是叶子节点且路径和等于目标值
            if (remainSum == 0) {
                // 找到一条有效路径，加入结果集
                // 注意：要 new 一个新的 ArrayList，不能直接加 path 引用
                result.add(new ArrayList<>(path));
            }
        } else {
            // 不是叶子节点，继续递归左右子树
            dfs(node.left, remainSum);
            dfs(node.right, remainSum);
        }

        // 3. 撤销选择：回溯，移除当前节点
        path.removeLast();
    }
}
```

---

### 代码详解

#### 1. 为什么用 `LinkedList` 存储路径？

```java
LinkedList<Integer> path = new LinkedList<>();
```

- `LinkedList` 支持 O(1) 时间在尾部添加/删除元素
- `path.add()` 在尾部添加
- `path.removeLast()` 在尾部删除（回溯用）
- 比 `ArrayList` 的 `remove(size-1)` 更高效

#### 2. 为什么要 `new ArrayList<>(path)`？

```java
result.add(new ArrayList<>(path));  // ✅ 正确
result.add(path);                    // ❌ 错误
```

**原因：** `path` 是同一个对象引用，后续回溯会修改它。

**错误示例：**
```java
// 如果直接加 path 引用：
result.add(path);  // 加入的是 path 的引用

// 后续回溯 path.removeLast()
// result 里的路径也会被修改！
// 最后 result 里全是空列表
```

**正确做法：** 拷贝一份当前路径的快照

#### 3. 为什么传递 `remainSum` 而不是累加 `currentSum`？

```java
// 方式 1：传递剩余和（推荐）
remainSum -= node.val;
if (remainSum == 0) { ... }

// 方式 2：传递当前和
currentSum += node.val;
if (currentSum == targetSum) { ... }
```

**两种方式都可以**，但传递剩余和有以下好处：
- 判断条件更直观：`remainSum == 0` 表示刚好凑够
- 避免在递归中传递额外的 `targetSum` 参数

## 四、复杂度分析

### 时间复杂度：O(N²)

- **遍历部分**：需要遍历所有节点，O(N)
- **拷贝路径部分**：找到有效路径时需要拷贝路径，路径长度最坏为 O(N)
- **总复杂度**：O(N) × O(N) = O(N²)

**最坏情况：** 树退化成链表，且每条路径都满足条件

### 空间复杂度：O(N)

- **递归栈空间**：树的高度，最坏 O(N)（链表），最好 O(logN)（平衡树）
- **路径存储**：O(N)（从根到叶子的路径长度）
- **总复杂度**：O(N)

## 六、易错点总结

### 易错点 1：忘记判断叶子节点

```java
// ❌ 错误：只要和为 0 就记录
if (remainSum == 0) {
    result.add(new ArrayList<>(path));
}

// ✅ 正确：必须是叶子节点且和为 0
if (node.left == null && node.right == null && remainSum == 0) {
    result.add(new ArrayList<>(path));
}
```

**原因：** 非叶子节点也可能 `remainSum == 0`，但路径还没结束

---

### 易错点 2：忘记回溯

```java
// ❌ 错误：没有移除当前节点
private void dfs(TreeNode node, int remainSum) {
    path.add(node.val);
    // ... 递归逻辑
    // 忘记写 path.removeLast()
}

// ✅ 正确：递归后要回溯
private void dfs(TreeNode node, int remainSum) {
    path.add(node.val);
    // ... 递归逻辑
    path.removeLast();  // 回溯
}
```

**后果：** 路径会包含所有访问过的节点，而不是当前路径

---

### 易错点 3：直接添加 path 引用

```java
// ❌ 错误：添加的是引用
result.add(path);

// ✅ 正确：添加的是拷贝
result.add(new ArrayList<>(path));
```

**后果：** `result` 里的路径会被后续回溯修改，最后全是空列表

---

### 易错点 4：叶子节点判断错误

```java
// ❌ 错误：判断条件不对
if (node.left != null || node.right != null) {
    // 这不是叶子节点
}

// ✅ 正确
if (node.left == null && node.right == null) {
    // 这是叶子节点
}
```

## 七、变体题目

### 变体 1：只需要判断是否存在路径（LeetCode 112）

```java
// 不需要记录路径，只需要返回 true/false
public boolean hasPathSum(TreeNode root, int targetSum) {
    if (root == null) {
        return false;
    }

    // 叶子节点且和为 0
    if (root.left == null && root.right == null) {
        return targetSum == root.val;
    }

    // 递归左右子树
    return hasPathSum(root.left, targetSum - root.val)
        || hasPathSum(root.right, targetSum - root.val);
}
```

---

### 变体 2：任意节点到任意节点的路径和（LeetCode 437）

```java
// 路径不需要从根节点开始，也不需要到叶子节点结束
// 但方向必须是向下的（父→子）

public int pathSum(TreeNode root, int targetSum) {
    if (root == null) {
        return 0;
    }

    // 以当前节点为起点的路径数
    int count = countPaths(root, targetSum);

    // 递归左右子树
    count += pathSum(root.left, targetSum);
    count += pathSum(root.right, targetSum);

    return count;
}

private int countPaths(TreeNode node, int remainSum) {
    if (node == null) {
        return 0;
    }

    int count = 0;
    if (remainSum == node.val) {
        count = 1;  // 找到一条路径
    }

    // 继续向下找（路径可以延伸）
    count += countPaths(node.left, remainSum - node.val);
    count += countPaths(node.right, remainSum - node.val);

    return count;
}
```

### 变体 3：最大路径和（LeetCode 124）

```java
// 路径可以从任意节点到任意节点，不一定经过根节点
// 求所有路径中和最大的

int maxSum = Integer.MIN_VALUE;

public int maxPathSum(TreeNode root) {
    dfs(root);
    return maxSum;
}

// 返回以当前节点为起点，向下的最大路径和
private int dfs(TreeNode node) {
    if (node == null) {
        return 0;
    }

    // 左右子树的最大贡献（负数就不要了）
    int leftGain = Math.max(0, dfs(node.left));
    int rightGain = Math.max(0, dfs(node.right));

    // 以当前节点为"转折点"的路径和
    int currentSum = node.val + leftGain + rightGain;
    maxSum = Math.max(maxSum, currentSum);

    // 返回当前节点能向上提供的最大贡献
    return node.val + Math.max(leftGain, rightGain);
}
```

## 八、面试答题话术

**面试官问：二叉树根节点到叶子节点路径和等于目标值，怎么找？**

答：这道题我用**前序遍历 + 回溯**来解决。

**思路是这样的：**

1. 前序遍历的顺序是根→左→右，天然符合"从根到叶子"的路径要求。

2. 遍历过程中，用一个 `path` 列表记录当前路径，用 `remainSum` 记录剩余目标和。

3. 对于每个节点：
   - 先加入路径，更新剩余和
   - 如果是叶子节点且剩余和为 0，说明找到一条有效路径，加入结果集
   - 如果不是叶子节点，继续递归左右子树
   - 递归返回后，从路径中移除当前节点（回溯）

4. 最后返回所有有效路径。

**时间复杂度是 O(N²)**，因为要遍历所有节点 N，找到有效路径时拷贝路径需要 O(N)。

**空间复杂度是 O(N)**，递归栈和路径存储各需要 O(N)。

**需要注意的易错点：**
1. 必须判断是叶子节点，不能只看和是否为 0
2. 加入结果集时要 `new ArrayList<>(path)` 拷贝，不能直接加引用
3. 递归后要回溯，移除当前节点

**代码核心部分：**
```java
path.add(node.val);
remainSum -= node.val;

if (node.left == null && node.right == null && remainSum == 0) {
    result.add(new ArrayList<>(path));
} else {
    dfs(node.left, remainSum);
    dfs(node.right, remainSum);
}

path.removeLast();  // 回溯
```
